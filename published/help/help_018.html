Onglet "Menuiseries" - "Performances des menuiseries"
  • Dans le cadre du présent outil pédagogique, les hypothèses suivantes sont retenues :
    • Les portes-fenêtres côté jardin sont en aluminium. La porte d’entrée ainsi que la porte d’accès entre le cellier et le garage sont en bois et isolées. L’ensemble des autres menuiseries extérieures sont en PVC.
    • Les classes de performances proposées correspondent aux caractéristiques techniques suivantes :
      • "Basse" :
        • PVC : Uw=1.60 W/(m².K) // Sw=0.42 // Tlw=0.53;
        • Alu : Uw=1.80 W/(m².K) // Sw=0.51 // Tlw=0.60.
      • "Standard" :
        • PVC : Uw=1.40 W/(m².K) // Sw=0.44 // Tlw=0.55;
        • Alu : Uw=1.60 W/(m².K) // Sw=0.53 // Tlw=0.62.
      • "Haute" :
        • PVC : Uw=1.30 W/(m².K) // Sw=0.49 // Tlw=0.60;
        • Alu : Uw=1.40 W/(m².K) // Sw=0.59 // Tlw=0.67.
    • Les caractéristiques techniques des portes pleines sont les suivantes :
      • Porte d’entrée : Uw=1.00 W/(m².K) ;
      • Porte d’accès entre le cellier et le garage : Uw=1.20 W/(m².K).

  • Les performances thermiques des menuiseries sont caractérisées par 3 paramètres principaux :
    • Le Uw, exprimé en W/(m².K), est le coefficient déperditif surfacique moyen d’une menuiserie, comprenant notamment les caractéristiques du cadre et du vitrage. Il définit la quantité de chaleur traversant 1 m² de menuiserie, pour une différence de température de 1°C. Plus le Uw est élevé, plus la menuiserie est déperditive.

    • Le Sw, exprimé sans unité, caractérise le facteur solaire de la menuiserie. Il représente la quantité de chaleur issue du rayonnement solaire qui traverse la menuiserie. Un Sw de 1 signifie que 100% du rayonnement solaire traverse la menuiserie, tandis qu’un Sw de 0 se traduit par une réflexion totale du rayonnement solaire par la menuiserie. En d’autres termes, plus le Sw est élevé, plus les apports de chaleur issus du rayonnement solaire sont importants. Ainsi les besoins de chauffage sont diminués, mais les besoins de froid augmentent.

    • Le Tlw, exprimé sans unité, caractérise la transmission lumineuse de la menuiserie et traduit la part d’éclairage naturel qui la traverse. Un Tlw de 0 signifie que la menuiserie est totalement occultante. Par conséquent, plus le Tlw est important, plus les besoins en éclairage artificiel diminuent.
