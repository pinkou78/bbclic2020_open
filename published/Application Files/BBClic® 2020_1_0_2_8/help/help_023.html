Aide & guide d'utilisation

Version v 0.1

Onglet "Planchers" - "Planchers bas" - "Type de plancher"
  • Ce menu permet de choisir le principe constructif du plancher bas. Le coefficient Up, exprimé en W/m².K, correspond aux déperditions surfaciques de cette paroi. Plus cette valeur est basse, plus le plancher est thermiquement isolant.

  • Les ponts thermiques périphériques de plancher bas dépendent du type de plancher retenu.

  • Le type de plancher bas a donc un impact sur les besoins de chauffage : plus le plancher est isolant, plus les besoins de chauffage sont faibles.

  • Dans la présente version de l’outil, tous les planchers bas sont sur vide sanitaire.

  • Dans le cas de la maison individuelle de plain-pied, passer d’un plancher hourdis PSE Up=0.27 W/m².K à Up=0.19 W/m².K permet de gagner 5 à 8 points sur le Bbio selon la zone climatique.
