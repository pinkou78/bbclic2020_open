Toiture

Toiture-terrasse

Mode simple

- Jonction plancher/façade
  • Ce paramètre représente les dispositions spécifiques qui peuvent être prises pour la réduction du pont thermique (L9) de jonction entre la toiture-terrasse et la façade.

  • Le type de planelle choisie influe sur la quantité de chaleur perdue au niveau de cette jonction. Plus la planelle est isolante, plus ce flux thermique baisse, en particulier lorsque la maçonnerie en façade est également isolante. Les rupteurs de ponts thermiques constituent une autre solution performante. L’ensemble de ces différents systèmes peut être combiné.

  • Le dispositif de réduction de ponts thermiques retenu a donc un impact sur les besoins bioclimatiques, le ratio moyen de ponts thermiqueset le confort d’été.

  • Dans le cadre de cette outil, la variation d'épaisseur d'isolation complémentaire en toiture et la végétalisation sont négligés dans la détermination des valeurs de ponts thermiques.
