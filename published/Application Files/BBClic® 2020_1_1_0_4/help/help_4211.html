Planchers

Plancher intermédiaire

Mode simple

- Type de plancher
  • Ce menu permet de choisir le principe constructif du plancher intermédiaire. Un plancher intermédiaire est un plancher délimitant deux volumes chauffés. Ainsi, cette paroi est considérée comme étant non déperditive.

  • Les ponts thermiques périphériques de plancher intermédiaire (de type L9) dépendent du type de plancher retenu, de son épaisseur et du dispositif de réduction de ponts thermiques associé. Ils dépendent également des caractéristiques thermiques de la façade qui sont choisis dans le menu correspondant.

  • Le type de plancher intermédiaire a un impact sur les besoins bioclimatiques, les ratios de ponts thermiques (moyen et L9) et le confort d’été.
